# Category and subcategory management pages

Add management pages for categories and subcategories so users can organize receipt items.

## Requirements

* **Category manager**: list, create, edit, delete categories
* **Subcategory manager**: list, create, edit, delete subcategories, each linked to a parent category
* Consistent with the app's existing table/form patterns
* Categories and subcategories should then be selectable when creating/editing receipt items (via combobox, not UUID)
