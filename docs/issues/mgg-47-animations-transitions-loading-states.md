# Animations, Transitions & Loading States

## Objective

Add fluid animations and smooth transitions for a polished, snappy UX.

## Tasks

- [ ] Configure Tailwind transitions and animations
- [ ] Add Framer Motion for complex animations
- [ ] Implement page transitions (route changes)
- [ ] Create loading skeletons for all data fetches
- [ ] Add optimistic UI updates (instant feedback)
- [ ] Implement micro-interactions:
  - Button hover/active states
  - Card hover effects
  - Input focus animations
  - Checkbox/radio animations
- [ ] Create smooth modal open/close animations
- [ ] Add toast notification animations
- [ ] Implement skeleton loaders (skeleton UI pattern)
- [ ] Add drag-and-drop animations (if applicable)
- [ ] Create loading spinner component
- [ ] Add progress indicators for multi-step forms
- [ ] Respect prefers-reduced-motion for accessibility

## Acceptance Criteria

* All animations smooth (60fps)
* No layout shift during loading
* Optimistic updates feel instant
* Respects motion preferences
* Loading states clear and helpful
